# Native Streamlit Option A

This package rebuilds the dashboard as a native Streamlit app instead of embedding an HTML file.

## Files
- `app.py`
- `requirements.txt`
- `middle_sea_violations.csv`
- `middle_sea_logo.jpg`

## Deploy
1. Upload all files to GitHub
2. Open Streamlit Community Cloud
3. Choose repo and branch
4. Main file path: `app.py`
5. Deploy
